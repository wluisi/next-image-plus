---
title: "SourceProps Props Table"
status: true
---

| Name   | Type     | Required | Description                                                                                                               |
| ------ | -------- | -------- | ------------------------------------------------------------------------------------------------------------------------- |
| src    | `string` | Yes      | The URL of the image source. The value must be a path string or a statically imported file. `<Source src="/image.jpg" />` |
| width  | `number` | Yes      | The width prop functions the same way as it does on the Next.js `<Image />` component.                                    |
| height | `number` | Yes      | The height prop functions the same way as it does on the Next.js `<Image />` component.                                   |
