## parseMediaQuery

Parses a media query to get it's min and max values as numbers.

### Parameters

| Parameter | Description                      | Type     | Status   |
| --------- | -------------------------------- | -------- | -------- |
| media     | The media query string to parse. | `string` | Required |

### Returns

`null`

An object with min and max values.

| Name   | Description |
| ------ | ----------- |
| `min`  | -           |
| `max`  | -           |
