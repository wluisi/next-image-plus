## getFallbackMediaQuery

Generate a fallback media query covering everything below the smallest min-width.

### Parameters

| Parameter | Description                 | Type               | Status   |
| --------- | --------------------------- | ------------------ | -------- |
| items     | Array of media query items. | `MediaQueryItem[]` | Required |

### Returns

`null`

A fallback media query item or null.

| Name    | Description |
| ------- | ----------- |
| `uuid`  | -           |
| `media` | -           |
