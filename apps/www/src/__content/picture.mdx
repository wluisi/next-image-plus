# Picture

The `<Picture>` component provides 3 components for creating responsive images:

- Picture
- Source
- Image

These more or less map the html elements that are used for construcing repsonsive images.

Under the hood, these components leverage the [Nextjs Image API](https://nextjs.org/docs/pages/api-reference/components/image#getimageprops).

Here's a summary of the props available for the Picture, Source, and Image components.

## Picture Props

| Prop          | Example                              | Type    | Status |
| ------------- | ------------------------------------ | ------- | ------ |
| preload       | `preload={true}`                     | Boolean | -      |
| fallbackMedia | `fallbackMedia="(max-width: 430px)"` | String  | -      |

## Source Props

@note: src will automatically get converted into a srcset value.

| Prop   | Example                                  | Type         | Status   |
| ------ | ---------------------------------------- | ------------ | -------- |
| src    | `src="/image.jpg"`                       | String       | Required |
| media  | `media="(max-width: 430px)"`             | String       | Required |
| sizes  | `sizes="(max-width: 768px) 100vw, 33vw"` | String       | -        |
| width  | `width={500}`                            | Integer (px) | Required |
| height | `height={500}`                           | Integer (px) | Required |

## Image Props

@note: src will automatically get converted into a srcset value.

| Prop   | Example                                  | Type         | Status   |
| ------ | ---------------------------------------- | ------------ | -------- |
